# 📦 Mini Logistics Tracker

A simple logistics tracking web application built using **Flask**, deployed with **Docker**, and monitored using **Prometheus and Grafana**.

This project was developed as my final project for Cloud Computing Internship.

---

## 🚀 Features

- Add, remove, and manage items
- Automatic quantity grouping (e.g., 2x banana)
- Search functionality
- Item summary panel (total + unique items)
- Modern and responsive UI
- Real-time monitoring with Prometheus and Grafana

---

## 🧰 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (embedded in Flask)
- **Backend:** Python Flask
- **Containerization:** Docker
- **Monitoring:** Prometheus, Grafana
- **CI/CD:** GitHub Actions
- **Orchestration (prepared):** Kubernetes (YAML files)
- **Infrastructure as Code (prepared):** Terraform
